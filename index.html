Equipment Request Audit

Select an Epic Excel export to find rows where a requested Special Needs term is not present in Equipment. The audit runs in this browser tab without sending, uploading, or saving the file.

Instructions

  1. Export the relevant Epic report as an .xlsx file.
  2. Select the report file here and select Run audit.
  3. The tool checks only Case #, Special Needs, and Equipment.

Current audit terms:

C-arm Airo Myosure fluid management Fluid management system Fluent Myosure NIM nerve Microscope Gamma Neoprobe Geiger Trunode
Matching is deterministic: exact matching first, then fixed-threshold fuzzy matching for typos. No synonyms, inference, abbreviation expansion, or clinical interpretation are used.

Room Rules Audit

Upload an Epic OR schedule export to check each case against room assignment rules. The audit runs locally in this browser tab — no file is uploaded or stored.

Instructions

  3. Violations are color-coded by tier: Tier 1 physical absolute, Tier 2 strong operational, Tier 3 service preference, Tier 4 surgeon preference, Tier 5 suggestion.
